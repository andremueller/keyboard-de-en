# keyboard-de-en

Mix keyboard layout for german software developers working with us layout and writing occasionally umlauts

Keyboard switching between American (US) layout and German (DE) layout drives me nuts and slows down everything. US Layout is perfect for software development. It helps simply typing special characters, different braces etc.

The idea is to just have a normal US layout. Umlauts will be hit by holding the ALT key and pressing the original place of ä,ö,ü, and ß keys.

## MacOS

- Open the given file `macos\EN_DE_Mix_Keylayout.keylayout` with the [Ukulele software](https://software.sil.org/ukelele/) and press "Install".
